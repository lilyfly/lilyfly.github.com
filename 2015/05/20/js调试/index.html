weinre调试

首先安装 Weinre

npm install -g weinre

安装完成之后，要在本地开启一个监听服务器，需要获取本机的局域网地址：

Mac 在终端执行 ipconfig getifaddr en0 命令。

Win 在命令行执行 ipconfig 命令。

ip为10.189.249.254

weinre —boundHost 10.189.249.254
